# Materiais da Tarefa (form_public_mro_task_materials)

Módulo Planejamento e Engenharia — aplicação do tipo Form.

Formulário de alocação de materiais e peças às tarefas. Define quais materiais são necessários para executar cada tarefa, com quantidades e part numbers.

## O que o usuário pode fazer

- Vincular materiais a uma tarefa.
- Especificar part number, descrição e quantidade necessária.
- Editar ou remover materiais alocados.

## Dados envolvidos

Tabela `public.mro_task_materials`. Relacionada com `mro_tasks` e `mro_materials`.
