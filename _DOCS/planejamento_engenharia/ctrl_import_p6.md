# Importador P6 Legado (ctrl_import_p6 / ctrl_import_p6_xlsx)

Módulo Planejamento e Engenharia — aplicação do tipo Form.

Versões legadas do importador do Primavera P6, mantidas para compatibilidade com formatos de exportação antigos.

## O que o usuário pode fazer

- Importar pacotes de manutenção de versões anteriores do P6.
- Converter arquivos CSV ou XLSX para a estrutura do MRO.

## Dados envolvidos

Tabelas `public.mro_projects`, `public.mro_wbs`, `public.mro_tasks`.
