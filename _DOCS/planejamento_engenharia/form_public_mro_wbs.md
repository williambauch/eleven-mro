# Cadastro de WBS (form_public_mro_wbs)

Módulo Planejamento e Engenharia — aplicação do tipo Form.

Formulário de cadastro da Estrutura Analítica do Projeto (EAP/WBS). Permite criar e editar os pacotes de trabalho que compõem a árvore do projeto.

## O que o usuário pode fazer

- Cadastrar novos pacotes de trabalho com código, nome e tipo de fase.
- Editar pacotes existentes.
- Vincular o WBS a um projeto.

## Dados envolvidos

Tabela `public.mro_wbs`.
