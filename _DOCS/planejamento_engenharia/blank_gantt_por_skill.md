# Gantt por Skill (blank_gantt_por_skill)

Módulo Planejamento e Engenharia — aplicação do tipo Blank.

Cronograma Gantt filtrado por especialidade técnica (skill). Permite ao planejamento visualizar as tarefas alocadas a cada especialidade e identificar gargalos de mão de obra.

## O que o usuário pode fazer

- Selecionar uma especialidade para visualizar suas tarefas no tempo.
- Identificar sobrecarga ou ociosidade de cada skill.

## Dados envolvidos

Tabelas `public.mro_tasks`, `public.mro_task_assignments` e `public.mro_skills`.
