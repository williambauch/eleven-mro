# Gantt Tracking (blank_gantt_tracking)

Módulo Planejamento e Engenharia — aplicação do tipo Blank.

Versão do Gantt focada no acompanhamento do progresso (tracking). Compara o planejado (baseline) com o realizado (actual) para cada tarefa, destacando desvios de cronograma.

## O que o usuário pode fazer

- Visualizar o progresso real versus o planejado.
- Identificar tarefas em atraso ou adiantadas.
- Acompanhar a evolução percentual de cada tarefa.

## Dados envolvidos

Tabelas `public.mro_tasks` e `public.mro_task_assignments`.
