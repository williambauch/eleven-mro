# Cadastro de Tarefas (form_public_mro_tasks)

Módulo Planejamento e Engenharia — aplicação do tipo Form.

Formulário de cadastro e edição de tarefas de manutenção. Permite definir todos os atributos de uma tarefa: código, nome, referências AMM, capítulo ATA, prazos, estimativas de horas, status, painéis de acesso e vínculo com a WBS.

## O que o usuário pode fazer

- Cadastrar novas tarefas (rotinas ou NRCs).
- Editar tarefas existentes.
- Definir referências técnicas (AMM, ATA, access panels).
- Ajustar prazos e horas estimadas.
- Vincular a tarefa a um projeto e WBS.

## Dados envolvidos

Tabela `public.mro_tasks`.
