# Gantt (gantt / blank_gantt / blank_gantt_chart / blank_gantt_skill)

Módulo Planejamento e Engenharia — aplicação do tipo Blank.

Visualizações do cronograma de tarefas no formato gráfico de Gantt. Permite acompanhar a linha do tempo das tarefas de manutenção com barras de duração, dependências e milestones. Inclui variações por especialidade (skill) e chart resumido.

## O que o usuário pode fazer

- Visualizar o cronograma do projeto em formato Gantt.
- Acompanhar datas de início, fim e duração das tarefas.
- Filtrar a visualização por especialidade técnica.

## Dados envolvidos

Tabela `public.mro_tasks` com dados de baseline e actual start/end.
