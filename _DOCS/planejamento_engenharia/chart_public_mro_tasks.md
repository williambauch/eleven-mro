# Gráfico de Tarefas (chart_public_mro_tasks)

Módulo Planejamento e Engenharia — aplicação do tipo Chart.

Gráfico com a distribuição das tarefas por status, projeto ou tipo. Fornece uma visão panorâmica do andamento do plano de manutenção.

## O que o usuário pode fazer

- Visualizar gráficos com contagem de tarefas por status.
- Acompanhar a evolução do plano ao longo do tempo.

## Dados envolvidos

Tabela `public.mro_tasks`.
