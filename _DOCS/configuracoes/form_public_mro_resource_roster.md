# Cadastro de Roster (form_public_mro_resource_roster)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Form.

Formulário de qualificação de recursos (roster). Permite associar skills e licenças ANAC a cada recurso.

## O que o usuário pode fazer

- Qualificar um recurso com uma skill específica.
- Registrar licenças e certificações.

## Dados envolvidos

Tabelas de qualificação (roster). Relaciona `mro_resources` e `mro_skills`.
