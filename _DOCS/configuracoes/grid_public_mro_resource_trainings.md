# Treinamentos (grid_public_mro_resource_trainings)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Grid.

Grade de consulta dos treinamentos realizados pelos recursos. Exibe histórico de treinamentos, datas de validade e certificações.

## O que o usuário pode fazer

- Consultar treinamentos de cada recurso.
- Identificar treinamentos vencidos ou a vencer.

## Dados envolvidos

Tabelas de treinamentos. Relaciona `mro_resources` com tipos de treinamento.
