# Recursos (grid_public_mro_resources)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Grid.

Grade de consulta dos recursos (mão de obra) cadastrados. Inclui o código do recurso, nome, tipo (LABOR, EQUIPMENT, etc.), skill e status ativo. Os recursos são importados do Primavera P6 ou cadastrados manualmente.

## O que o usuário pode fazer

- Visualizar todos os recursos disponíveis.
- Consultar tipo, skill e status de cada recurso.

## Dados envolvidos

Tabela `public.mro_resources`.
