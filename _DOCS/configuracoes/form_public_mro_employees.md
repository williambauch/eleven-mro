# Cadastro de Colaboradores (form_public_mro_employees)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Form.

Formulário de cadastro de colaboradores. Permite registrar dados pessoais, matrícula, código ANAC, especialidade, vínculo com usuário do sistema e permissões (inspetor, supervisor).

## O que o usuário pode fazer

- Cadastrar novos colaboradores.
- Editar colaboradores existentes.
- Definir se é inspetor, supervisor ou está ativo.

## Dados envolvidos

Tabela `public.mro_employees`.
