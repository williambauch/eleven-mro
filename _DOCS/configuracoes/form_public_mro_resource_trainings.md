# Cadastro de Treinamentos (form_public_mro_resource_trainings)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Form.

Formulário de registro de treinamentos dos recursos. Permite associar um treinamento a um mecânico, informando data de realização e validade.

## O que o usuário pode fazer

- Registrar treinamentos realizados por cada mecânico.
- Informar data de validade para controle de recorrência.

## Dados envolvidos

Tabelas de treinamentos.
