# Cadastro de Tipos de Treinamento (form_public_mro_training_types)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Form.

Formulário de cadastro de tipos de treinamento. Permite criar e editar os tipos de treinamento que podem ser associados aos recursos.

## O que o usuário pode fazer

- Cadastrar novos tipos de treinamento.
- Editar tipos existentes.

## Dados envolvidos

Tabela de tipos de treinamento.
