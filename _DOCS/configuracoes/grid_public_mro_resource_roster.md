# Roster (grid_public_mro_resource_roster)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Grid.

Grade de consulta da matriz de qualificação (roster). Exibe o cruzamento entre recursos, suas qualificações, licenças ANAC e treinamentos, permitindo verificar se um mecânico está apto a executar determinada tarefa.

## O que o usuário pode fazer

- Consultar a matriz de qualificação dos mecânicos.
- Verificar licenças, habilidades e treinamentos de cada recurso.

## Dados envolvidos

Tabelas de roster/qualificação. Relaciona `mro_resources`, `mro_skills` e treinamentos.
