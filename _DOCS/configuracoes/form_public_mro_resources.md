# Cadastro de Recursos (form_public_mro_resources)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Form.

Formulário de cadastro de recursos. Permite criar e editar recursos de mão de obra, definindo nome, tipo (LABOR, EQUIPMENT), skill, identificador P6 e status.

## O que o usuário pode fazer

- Cadastrar novos recursos.
- Editar recursos existentes.
- Importar dados do P6 (p6_rsrc_id).

## Dados envolvidos

Tabela `public.mro_resources`.
