# Cadastro de Skills (form_public_mro_skills)

Módulo Configurações / Tabelas Básicas — aplicação do tipo Form.

Formulário de cadastro de especialidades técnicas. Permite criar skills com código, descrição e tipo de licença ANAC.

## O que o usuário pode fazer

- Cadastrar novas especialidades.
- Editar especialidades existentes.

## Dados envolvidos

Tabela `public.mro_skills`.
