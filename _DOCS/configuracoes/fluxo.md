# Fluxo — Configurações / Tabelas Básicas

Módulo de cadastros base do sistema. Mantém o registro de colaboradores e mecânicos, suas habilitações (skills/especialidades) e dados de RH necessários para a matriz de qualificação e alocação de tarefas.

## Aplicações do módulo

- [Colaboradores e Mecânicos (grid_public_mro_employees)](grid_public_mro_employees.md) — cadastro de funcionários com dados de registro, habilitação e permissões.
- [Especialidades / Skills (grid_public_mro_skills)](grid_public_mro_skills.md) — cadastro de especialidades técnicas e licenças ANAC.
- [Cadastro de Colaboradores (form_public_mro_employees)](form_public_mro_employees.md) — formulário de cadastro de colaboradores.
- [Cadastro de Skills (form_public_mro_skills)](form_public_mro_skills.md) — formulário de cadastro de especialidades.
- [Recursos (grid_public_mro_resources)](grid_public_mro_resources.md) — consulta de recursos (mão de obra) do sistema.
- [Cadastro de Recursos (form_public_mro_resources)](form_public_mro_resources.md) — formulário de cadastro de recursos.
- [Roster (grid_public_mro_resource_roster)](grid_public_mro_resource_roster.md) — consulta da matriz de qualificação (roster).
- [Cadastro de Roster (form_public_mro_resource_roster)](form_public_mro_resource_roster.md) — formulário de qualificação de recursos.
- [Treinamentos (grid_public_mro_resource_trainings)](grid_public_mro_resource_trainings.md) — consulta de treinamentos dos recursos.
- [Cadastro de Treinamentos (form_public_mro_resource_trainings)](form_public_mro_resource_trainings.md) — formulário de registro de treinamentos.
- [Tipos de Treinamento (grid_public_mro_training_types)](grid_public_mro_training_types.md) — consulta de tipos de treinamento.
- [Cadastro de Tipos de Treinamento (form_public_mro_training_types)](form_public_mro_training_types.md) — formulário de tipos de treinamento.
