# Movimentações (form_public_mro_tool_movements)

Módulo Logística e Ferramentaria — aplicação do tipo Form.

Formulário de registro manual de movimentações de ferramentas. Permite registrar check-outs e check-ins fora do terminal de bipagem, para situações excepcionais.

## O que o usuário pode fazer

- Registrar manualmente a retirada de uma ferramenta.
- Registrar a devolução e a condição de retorno.

## Dados envolvidos

Tabela `public.mro_tool_movements`.
