# Incidentes com Ferramentas (form_public_mro_tool_incidents)

Módulo Logística e Ferramentaria — aplicação do tipo Form.

Formulário de registro de incidentes envolvendo ferramentas, como danos, avarias ou extravios. Faz parte do fluxo SGSO (Sistema de Gestão da Segurança Operacional).

## O que o usuário pode fazer

- Registrar um incidente com uma ferramenta.
- Descrever o ocorrido e as ações corretivas tomadas.

## Dados envolvidos

Tabela `public.mro_tool_incidents`.
