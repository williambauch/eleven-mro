# Cadastro de Ferramentas (form_public_mro_tools)

Módulo Logística e Ferramentaria — aplicação do tipo Form.

Formulário de cadastro e edição de ferramentas. Permite registrar part number, número de série, descrição, localização, status e datas de calibração.

## O que o usuário pode fazer

- Cadastrar novas ferramentas no sistema.
- Editar dados de ferramentas existentes.
- Registrar datas de calibração e manter o histórico.

## Dados envolvidos

Tabela `public.mro_tools`.
