# Grid de Incidentes (grid_public_mro_tool_incidents)

Módulo Logística e Ferramentaria — aplicação do tipo Grid.

Grade de consulta de incidentes com ferramentas. Permite auditar o histórico de ocorrências e ações corretivas.

## O que o usuário pode fazer

- Visualizar incidentes registrados.
- Filtrar por ferramenta, período ou tipo de incidente.

## Dados envolvidos

Tabela `public.mro_tool_incidents`.
