# Controle 2FA (sec_control_2fa)

Módulo Security — aplicação do tipo Form.

Tela administrativa de gerenciamento da autenticação em dois fatores para todos os usuários. Permite ao administrador visualizar quem possui 2FA ativo e resetar a configuração quando necessário.

## O que o usuário pode fazer

- Visualizar o status de 2FA dos usuários.
- Resetar o 2FA de um usuário específico (caso ele tenha perdido o acesso ao seu método de autenticação).

## Dados envolvidos

Tabela `public.sec_users`.
