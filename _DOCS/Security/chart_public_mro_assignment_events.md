# Eventos de Assignment (chart_public_mro_assignment_events)

Módulo Security — aplicação do tipo Chart.

Gráfico de eventos de alocação de tarefas. Exibe dados visuais sobre as movimentações de task assignments, útil para auditoria e acompanhamento gerencial.

## O que o usuário pode fazer

- Visualizar gráficos com dados de eventos de alocação.
- Acompanhar tendências e histórico de movimentações.

## Dados envolvidos

Tabela de eventos de task assignments (log de alterações de status).
