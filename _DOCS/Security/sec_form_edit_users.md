# Edição de Usuários (sec_form_edit_users)

Módulo Security — aplicação do tipo Form.

Tela de edição de usuários existentes. Permite alterar dados cadastrais, redefinir senha, ativar/desativar o usuário e ajustar privilégios de administrador.

## O que o usuário pode fazer

- Editar nome, e-mail e login do usuário.
- Ativar ou desativar a conta.
- Conceder ou remover privilégio de administrador.
- Redefinir a senha do usuário.

## Dados envolvidos

Tabela `public.sec_users`.
