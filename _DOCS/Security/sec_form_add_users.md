# Formulário de Usuários (sec_form_add_users)

Módulo Security — aplicação do tipo Form.

Tela de cadastro de novos usuários no sistema. Permite criar uma conta com login, nome, e-mail, senha e definir se o usuário terá privilégios de administrador.

## O que o usuário pode fazer

- Cadastrar novos usuários com dados pessoais e de acesso.
- Definir se o usuário é administrador.
- Após salvar, o usuário estará disponível para associação a grupos.

## Dados envolvidos

Tabela `public.sec_users`.
