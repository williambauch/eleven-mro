# Fluxo — Security

Módulo de segurança e administração do sistema. Gerencia usuários, grupos, permissões, aplicações, autenticação em dois fatores e configurações globais.

Acesso restrito a administradores do sistema. As telas deste módulo são acessadas através do menu de segurança (`sec_menu`) e controlam quem pode acessar o quê dentro da plataforma.

## Aplicações do módulo

- [Usuários (sec_grid_sec_users)](sec_grid_sec_users.md) — listagem e gerenciamento de usuários do sistema.
- [Aplicações (sec_grid_sec_apps)](sec_grid_sec_apps.md) — consulta das aplicações registradas no sistema.
- [Grupos (sec_grid_sec_groups)](sec_grid_sec_groups.md) — cadastro e listagem de grupos de acesso.
- [Usuários x Grupos (sec_grid_sec_users_groups)](sec_grid_sec_users_groups.md) — associação de usuários a grupos e controle de permissões.
- [Grupos x Aplicações (sec_search_sec_groups)](sec_search_sec_groups.md) — consulta de permissões de um grupo sobre as aplicações.
- [Sincronizar Aplicações (sec_sync_apps)](sec_sync_apps.md) — sincronização das aplicações do disco com o registro de segurança.
- [Autenticação em 2 Fatores (sec_add_2fa)](sec_add_2fa.md) — configuração e ativação da verificação em duas etapas.
- [Configurações do Sistema (sec_settings)](sec_settings.md) — ajustes globais de segurança, e-mail, 2FA e aparência.
- [Login (login)](login.md) — tela de autenticação do sistema.
- [Formulário de Usuários (sec_form_add_users)](sec_form_add_users.md) — cadastro de novos usuários.
- [Edição de Usuários (sec_form_edit_users)](sec_form_edit_users.md) — edição de usuários existentes.
- [Formulário de Aplicações (sec_form_sec_apps)](sec_form_sec_apps.md) — cadastro manual de aplicações.
- [Formulário de Grupos (sec_form_sec_groups)](sec_form_sec_groups.md) — cadastro de grupos de acesso.
- [Grupos x Aplicações (sec_form_sec_groups_apps)](sec_form_sec_groups_apps.md) — definição de permissões de grupos sobre aplicações.
- [Meus Dados (sec_my_info)](sec_my_info.md) — informações do perfil do usuário logado.
- [Alterar Senha (sec_change_pswd)](sec_change_pswd.md) — troca de senha do usuário.
- [Recuperar Senha (sec_retrieve_pswd)](sec_retrieve_pswd.md) — recuperação de senha esquecida.
- [Controle 2FA (sec_control_2fa)](sec_control_2fa.md) — gerenciamento de autenticação em dois fatores.
- [Eventos de Assignment (chart_public_mro_assignment_events)](chart_public_mro_assignment_events.md) — gráfico de eventos de alocação de tarefas.
