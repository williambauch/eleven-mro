# Meus Dados (sec_my_info)

Módulo Security — aplicação do tipo Form.

Tela de visualização e edição das informações do próprio perfil do usuário logado. Permite alterar dados pessoais como nome, e-mail e foto.

## O que o usuário pode fazer

- Visualizar seus dados cadastrais.
- Alterar nome, e-mail e foto do perfil.
- As alterações são salvas para o usuário logado.

## Dados envolvidos

Tabela `public.sec_users`.
