# Formulário de Aplicações (sec_form_sec_apps)

Módulo Security — aplicação do tipo Form.

Tela de cadastro manual de aplicações no sistema de segurança. Permite registrar aplicações que não foram detectadas automaticamente pela sincronização.

## O que o usuário pode fazer

- Cadastrar novas aplicações informando nome técnico e descrição.
- Editar aplicações existentes.

## Dados envolvidos

Tabela `public.sec_apps`.
