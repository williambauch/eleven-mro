# Formulário de Grupos (sec_form_sec_groups)

Módulo Security — aplicação do tipo Form.

Tela de cadastro de grupos de acesso. Cada grupo pode conter múltiplos usuários e ter permissões específicas sobre as aplicações do sistema.

## O que o usuário pode fazer

- Criar novos grupos com descrição.
- Editar grupos existentes.
- Excluir grupos (desde que não haja vínculos ativos).

## Dados envolvidos

Tabela `public.sec_groups`.
