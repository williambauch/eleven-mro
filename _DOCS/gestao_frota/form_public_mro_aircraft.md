# Cadastro de Aeronaves (form_public_mro_aircraft)

Módulo Gestão de Frota e Projetos — aplicação do tipo Form.

Formulário de cadastro e edição de aeronaves. Permite registrar matrícula, modelo, número de série (MSN) e cliente proprietário.

## O que o usuário pode fazer

- Cadastrar novas aeronaves no sistema.
- Editar dados de aeronaves existentes.

## Dados envolvidos

Tabela `public.mro_aircraft`.
