# Cadastro de Projetos (form_public_mro_projects)

Módulo Gestão de Frota e Projetos — aplicação do tipo Form.

Formulário de cadastro e edição de projetos de manutenção. Permite definir nome, aeronave associada, período, status e identificador de integração com o Primavera P6.

## O que o usuário pode fazer

- Criar novos projetos de manutenção.
- Editar projetos existentes.
- Vincular o projeto a uma aeronave da frota.

## Dados envolvidos

Tabela `public.mro_projects`. Relacionada com `public.mro_aircraft`.
