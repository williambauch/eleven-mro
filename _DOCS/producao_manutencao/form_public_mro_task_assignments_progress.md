# Alocações — Em Andamento (form_public_mro_task_assignments_progress)

Módulo Produção e Manutenção — aplicação do tipo Form.

Formulário de tarefas em execução pelos mecânicos. É a segunda aba do Painel do Supervisor, exibindo as alocações com status IN_PROGRESS.

## O que o usuário pode fazer

- Acompanhar as tarefas que estão sendo executadas no momento.
- Visualizar o tempo decorrido de cada alocação.
- Intervir caso necessário (remanejar, pausar ou encerrar).

## Dados envolvidos

Tabela `public.mro_task_assignments` com status IN_PROGRESS.
