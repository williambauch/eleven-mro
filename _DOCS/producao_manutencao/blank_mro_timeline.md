# Timeline (blank_mro_timeline)

Módulo Produção e Manutenção — aplicação do tipo Blank.

Linha do tempo visual das tarefas de um projeto. Exibe a sequência cronológica das tarefas com seus status, permitindo uma visão rápida do histórico de execução.

## O que o usuário pode fazer

- Visualizar a linha do tempo de tarefas por projeto.
- Acompanhar datas de início e fim das atividades.
- Identificar a sequência de execução.

## Dados envolvidos

Tabela `public.mro_tasks` com dados de actual start/end.
