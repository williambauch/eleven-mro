# Kanban (blank_kanban_board)

Módulo Produção e Manutenção — aplicação do tipo Blank.

Quadro Kanban para acompanhamento visual do fluxo de tarefas. Organiza as tarefas em colunas por status (planejado, em andamento, bloqueado, concluído), permitindo arrastar e soltar cards para atualizar o status.

## O que o usuário pode fazer

- Visualizar tarefas em colunas de status.
- Mover tarefas entre colunas para atualizar o progresso.
- Identificar rapidamente gargalos no fluxo.

## Dados envolvidos

Tabela `public.mro_task_assignments` e `public.mro_tasks`.
