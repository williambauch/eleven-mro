# Anexos de Tarefa (form_public_mro_task_attachments)

Módulo Produção e Manutenção — aplicação do tipo Form.

Tela de gestão de documentos e anexos das tarefas de manutenção. Permite vincular arquivos (fotos, PDFs, desenhos técnicos) a uma tarefa, compondo o dossiê digital da manutenção.

## O que o usuário pode fazer

- Anexar arquivos a uma tarefa.
- Visualizar e baixar anexos existentes.
- Remover anexos quando necessário.

## Dados envolvidos

Tabelas de GED (Gestão Eletrônica de Documentos). Relacionada com `mro_tasks`.
