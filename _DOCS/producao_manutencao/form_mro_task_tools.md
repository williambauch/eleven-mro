# Ferramentas da Tarefa (form_mro_task_tools)

Módulo Produção e Manutenção — aplicação do tipo Form.

Formulário de vínculo de ferramentas necessárias a cada tarefa. Define o empenho planejado de ferramentas que o mecânico precisará para executar o serviço.

## O que o usuário pode fazer

- Vincular ferramentas a uma tarefa.
- Especificar part number e quantidade necessária.

## Dados envolvidos

Tabela `public.mro_task_tools`. Relacionada com `mro_tasks` e `mro_tools`.
