# Grid de Ferramentas da Tarefa (grid_mro_task_tools)

Módulo Produção e Manutenção — aplicação do tipo Grid.

Grade de consulta de ferramentas vinculadas a tarefas. Exibe o empenho planejado de ferramentas para cada tarefa.

## O que o usuário pode fazer

- Consultar ferramentas planejadas por tarefa.
- Verificar se a ferramenta está disponível no estoque.

## Dados envolvidos

Tabela `public.mro_task_tools`.
