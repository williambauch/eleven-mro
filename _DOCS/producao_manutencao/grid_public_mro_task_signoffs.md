# Grid de Assinaturas (grid_public_mro_task_signoffs)

Módulo Produção e Manutenção — aplicação do tipo Grid.

Grade de consulta de assinaturas digitais registradas no sistema. Permite auditAR quem assinou cada tarefa, quando e em qual projeto.

## O que o usuário pode fazer

- Visualizar o histórico de assinaturas.
- Filtrar por tarefa, inspetor ou período.

## Dados envolvidos

Tabela `public.mro_task_signoffs`.
