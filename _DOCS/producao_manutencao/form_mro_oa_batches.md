# Lotes de O&A (form_mro_oa_batches)

Módulo Produção e Manutenção — aplicação do tipo Form.

Formulário de criação e edição de lotes de Over & Above. Permite agrupar NRCs em lotes para submissão à aprovação do cliente.

## O que o usuário pode fazer

- Criar novos lotes de O&A.
- Adicionar NRCs ao lote.
- Submeter o lote para aprovação do cliente.
- Acompanhar o status de aprovação.

## Dados envolvidos

Tabela `public.mro_oa_batches`. Relacionada com `public.mro_tasks`.
