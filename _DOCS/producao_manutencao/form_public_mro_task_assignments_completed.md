# Alocações — Concluídas (form_public_mro_task_assignments_completed)

Módulo Produção e Manutenção — aplicação do tipo Form.

Formulário de tarefas finalizadas pelos mecânicos. É a quarta aba do Painel do Supervisor, exibindo alocações com status COMPLETED.

## O que o usuário pode fazer

- Visualizar tarefas concluídas pela equipe.
- Verificar horas reais apontadas versus horas planejadas.
- Acompanhar as assinaturas de qualidade pendentes.

## Dados envolvidos

Tabela `public.mro_task_assignments` com status COMPLETED.
