# JEC PDF (pdf_jec)

Módulo Produção e Manutenção — aplicação do tipo Blank.

Geração de PDF do cartão JEC (Job Execution Card). Documento que registra a execução da tarefa com campos para dados reais, horas apontadas e assinaturas.

## Dados envolvidos

Tabelas `public.mro_tasks`, `public.mro_task_assignments` e `public.mro_timesheet`.
