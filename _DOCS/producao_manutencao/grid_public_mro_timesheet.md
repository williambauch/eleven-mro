# Grid de Timesheet (grid_public_mro_timesheet)

Módulo Produção e Manutenção — aplicação do tipo Grid.

Grade de consulta de todos os apontamentos de horas do sistema. Permite auditar as horas trabalhadas por mecânico, tarefa e projeto.

## O que o usuário pode fazer

- Visualizar todos os registros de timesheet.
- Filtrar por mecânico, tarefa, projeto ou período.
- Acompanhar a produtividade da equipe.

## Dados envolvidos

Tabela `public.mro_timesheet`.
