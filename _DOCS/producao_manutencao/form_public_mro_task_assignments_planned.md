# Alocações — A Distribuir (form_public_mro_task_assignments_planned)

Módulo Produção e Manutenção — aplicação do tipo Form.

Formulário de tarefas planejadas que ainda não foram alocadas a mecânicos. É a primeira aba do Painel do Supervisor, onde o supervisor visualiza as tarefas liberadas e designa os mecânicos para executá-las.

## O que o usuário pode fazer

- Visualizar tarefas no status NOT_STARTED.
- Alocar mecânicos disponíveis para cada tarefa.
- Definir a especialidade e horas planejadas.

## Dados envolvidos

Tabela `public.mro_task_assignments` com status NOT_STARTED.
